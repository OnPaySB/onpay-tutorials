Topik 9 : Pengurusan Affiliate ( Media + Trafik + Komisyen )


9.1 Media Affiliate Lokasi ( Affiliate > Media )


Panduan

Tambah Item Baru

1. Sila tekan butang Tambah Item Baru
2. Isikan kotak dengan maklumat yang sesuai.
3. Tekan Butang Simpan

Kemaskini Item

1. Sila tekan butang Kemaskini
2. Isikan kotak dengan maklumat yang sesuai.
3. Tekan Butang Simpan

Padam Item

1. Sila tekan butang Padam
2. Tekan Butang Ya

9.2 Trafik Lokasi ( Affiliate > Trafik )


Panduan
Kegunaan Senarai Trafik

Matlamat Senarai Trafik Ejen hanyalah untuk memaparkan senarai trafik yang datang dari ejen affiliate.
Antara maklumat yang dipaparkan adalah:

1. Bil
2. ID Ejen
3. Tarikh Dan Masa Trafik Masuk
4. Alamat IP sumber trafik
5. Sumber Trafik (dari laman mana prospek akses link tersebut)
- "Direct" dipaparkan sekiranya prospek menulis sendiri alamat url affiliate. - "URL" dipaparkan sekiranya link affiliate diakses dari halaman lain. Contohnya : www.facebook.com

Manipulasi Rekod Trafik

Tiada proses manipulasi rekod trafik adalah dibenarkan.

9.3 Komisyen Lokasi ( Affiliate > Komisyen )


Panduan

1. Buat Pembayaran Komisyen
-Tekan butang $ pada rekod ejen yang diingini.

2. Paparkan Maklumat Pembayaran Komisyen
- Tekan butang mata.

3. Muat Turun Invois Pembayaran
- Tekan Butang kertas pada kolum tindakan

4. Batal Pembayaran Komisyen
- Tekan butang batal pada kolum tindakan.
